====================
Publish your session
====================

Nucleome Browser supports users to publish their sessions/data servers onto our `Gallery website <https://gallery.nucleome.org>`_.


.. figure:: img/figures_chapter_8/NB_gallery_homepage.png
    :align: center
    :figwidth: 640px

    Users can publish their sessions/data servers by sending a pull request to https://github.com/nucleome/sessions. The Gallery web portal will automatically update and show published sessions.

Users can follow the instructions on this `GitHub repo <https://github.com/nucleome/sessions>`_ to prepare a public session with version control.
