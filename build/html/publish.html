Publish your session¶

Nucleome Browser Gallery¶

Nucleome Browser supports users to publish their sessions/data servers onto our Gallery website.

Fig. 65 Users can publish their sessions/data servers by sending a pull request to https://github.com/nucleome/sessions. The Gallery web portal will automatically update and show published sessions.

Users can follow the instructions on this GitHub repo to prepare a public session with version control.
